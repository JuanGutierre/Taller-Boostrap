Mi Portafolio Personal
Este proyecto es un sitio web de portafolio personal que muestra mis habilidades, proyectos y proporciona una forma de contacto. El sitio web está diseñado para ser responsive y atractivo, utilizando componentes de Bootstrap para una interfaz de usuario moderna.

Tecnologías Utilizadas
HTML5
CSS3
JavaScript
Bootstrap 5
Google Fonts 

Características
Diseño responsive
Navbar con enlaces a secciones
Sección "Sobre Mí" con modal para mostrar habilidades
Sección de proyectos con tarjetas
Formulario de contacto
Footer con información de derechos de autor
